30  \ref LOEntityListRef. It is possible to create a group from entities
31  that are already in a document, as well as from entities that are not
32  yet in a document. For entities that are already in a document, they can
33  only be grouped if they all belong to shared layers, or all belong to
34  non-shared layers on the same page. If the entities are in a document,

36  hierarchy.
37 @param[out] group The group object.

40 - \ref SU_ERROR_NONE on success

43 - \ref SU_ERROR_INVALID_INPUT if entity_list does not refer to a valid object
44 - \ref SU_ERROR_GENERIC if entity_list contains fewer than two entities
45 - \ref SU_ERROR_GENERIC if entity_list contains entities that belong to
